# Screenshots

The README references three images in this folder. Capture them from the app
(`⌘⇧4` then Space then click the window on macOS) and save with these exact names:

| Filename | What to capture |
|----------|-----------------|
| `main.png` | The default view — editor on the left, color-coded preview on the right (dark theme). |
| `substitutions.png` | Click a chord (e.g. `Cmaj7`) so the substitution menu with the piano diagram is open. |
| `themes.png` | Switch the theme (◐ Theme → Terminal / Sepia / Pastel) to show the theming. |
| `commands.png` | The command palette open (⌘K). |

PNG is preferred. After adding the files, `git add -A && git commit && git push`.
